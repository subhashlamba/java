# Java Regular Expression for mobile number 

## Expression: ^(\+?\d{2})?(\s+)?(\d{10})\b

### Valid Mobile No:
* 0123456789
* +91 0123456789
* 91 0123456789
* +910123456789
* +91          0123456789
 
### Invalid Mobile No
*  0123456789000
* 0123 456 789
